# CITILEX ASIA

Landing page resmi CITILEX ASIA.

## About

CITILEX ASIA menyediakan produksi **kaos custom untuk event, gathering, seminar, dan kebutuhan perusahaan** dengan fokus pada kualitas produksi, pilihan material, dan proses pemesanan yang terstruktur.

Website ini dibuat sebagai landing page conversion untuk kebutuhan Google Ads, SEO, dan customer inquiry melalui WhatsApp.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Vercel Deployment
- Google Tag Manager
- Google Analytics 4
- Google Ads Conversion Tracking

## Features

- Responsive landing page
- SEO metadata optimization
- Open Graph & social sharing metadata
- Structured Data (JSON-LD)
- Pricing section
- Product showcase
- FAQ section
- WhatsApp CTA conversion tracking
- Google Ads conversion tracking
- GA4 visitor analytics
- Search Console ready
